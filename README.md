# aiCompanyElectroProm
Компания «ЭлектроПром» специализируется на производстве бытовой электроники (кухонная техника, климатические устройства, аудиотехника). Текущий производственный процесс включает этапы: планирование и закупка компонентов, сборка на автоматизированных линиях с ручными операциями, функциональное тестирование, упаковка.
